# Installation instructions

1. Checkout to `~/dotfiles` 
1. cd to `~/dotfiles`
1. install GNU stow `pacman -S stow`
1. Run the relevant `stow` commands you wish to symlink


# Stow commands
```sh
stow bspwm
stow fish
stow git
stow mpd
stow ncmpcpp
stow nvim
stow thefuck
stow tmux
stow vim
stow alacritty
sudo stow xorg -t /etc/X11/
```
